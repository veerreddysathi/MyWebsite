A Quality Criteria Based Evaluation of Topic Models (Master Thesis)

Blekinge Institute of Technology, Karlskrona, Sweden

January, 2016 - October, 2016

Our study investigates the possibility of the application of Machine Learning Topic Models for Software test case prioritisation. During the testing phase, in numerous situations, test cases can be so many that executing each and every test case is practically impossible, as there may be many constraints. This causes the testers to prioritise the functions that are to be tested. This is where the ability of topic models can be exploited. Topic models are unsupervised machine learning algorithms that can explore large corpora of data, and classify them by identifying the hidden thematic structure in those corpora. Using topic models for test case prioritisation can save a lot of time and resources. In our study, we provide an overview of the amount of research that has been done in relation to topic models. We uncover various quality criteria, evaluation methods, and metrics that can be used to evaluate the topic models. Furthermore, we also compare the performance of two topic models that are optimised for different quality criteria, on a particular interpretability task, and thereby determine the topic model that produces the best results for that task.

A Trust Based ADHOC On Demand Routing Protocol for MANETs (Bachelor Thesis)

Jawaharlal Nehru Technological University, Kakinada, India

July, 2014 - October, 2014

This project was a replicative study of the research done by Naveen Kumar Gupta, Kavita Pandey; in which they propose a new routing protocol for MANETs with enhanced performance, and security in terms of data transmission between nodes. In our project, we present the results of trust based Ad hoc On Demand Routing protocol for MANETs by running simulations of data transmission between several nodes. The algorithm works on the concept of honest value which is calculated on the concept of hop and trust to protect the network from malicious node. The performance of the proposed protocol is analyzed using throughput, number of drop packets, packet delivery ratio and number of received packets with the variation of number of nodes, speed and simulation time. Results showed that the proposed method indeed has a better performance and enhances the security in the network.
